Задача которую решали: Нужно было вызывать асинхронную ф-ю аля синхронно

есть источник асинхронных событий, например

1const task = async <T>(value: T) => { 2 await new Promise((r) => setTimeout(r, 100 * Math.random())); 3 console.log(value); 4};

если их вызвать одновременно то получим случайный набор чисел

1await Promise.all([ 2 task(1), 3 task(2), 4 task(3), 5 task(4), 6]); 7 8// 4 1 3 2

необходимо написать AsyncQueue которая сохранит последовательность

1const queue = new AsyncQueue(); 2 3await Promise.all([ 4 queue.add(() => task(1)), 5 queue.add(() => task(2)), 6 queue.add(() => task(3)), 7 queue.add(() => task(4)), 8]); 9 10// 1 2 3 4
